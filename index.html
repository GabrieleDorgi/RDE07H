Branches e Hospedagem

Branches

Branches são ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • novas funcionalidades;
  • correção de problemas;
  • alteração de código.

Tipos de Branches:

Branch Master ou Main:
Braço principal de um projeto em um repositório.
Branch Secundária:
Braço secundário de um projeto em um repositório.

Como adicionar Branches:

  1. No GitHub Desktop, clicar em Branch;
  2. Clicar em New Branch;
  3. Colocar o nome da Branch;
  4. Clicar em Create Branch;
  5. Clicar em Publish Branch, para mandar para o servidor;

Hospedagem

A Hospedagem pode ser feita de modo gratuito pelo GitHub Pages. Basta criar uma conta e um repositório de acesso público e sua URL será:"https://nomeusuario.github.io/repositorio"

Como fazer a Hospedagem:

  1. Abrir o repositório no site do GitHub;
  2. Clicar em configurações;
  3. Tópico GitHub Pages(o repositório deve estar público);
  4. escolher a branch e salvar.
